Idha pana bhikkhave ekacco puggalo ummujjitvà tiõõo hoti pàragato thale tiññhati bràhmaõo.

(2) Katha勺a bhikkhave puggalo ummujjitvà nimujjati: idha bhikkhave ekacco puggalo ummujjati "sàdhu saddhà kusalesu dhammesu, sàdhu hiri kusalesu dhammesu, sàdhu ottappaü kusalesu dhammesu, sàdhu viriyaü kusalesu dhammesu, sàdhu pa中à kusalesu dhammeså"ti tassa sà saddhà neva tiññhati, no vaóóhati, hàyateva.Tassa sà hiri neva tiññhati,no vaóóhati,hàyateva. Tassa taü ottappaü neva tiññhati, no vaóóhati, hàyateva. Tassa taü viriyaü neva tiññhati, no vaóóhati, hàyateva. Tassa sà pa中à neva tiññhati, no vaóóhati, hàyateva. Evaü kho bhikkhave puggalo ummujjitvà nimujjati.

(3) Katha勺a bhikkhave puggalo ummujjitvà ñhito hoti: [PTS Page 012] idha bhikkhave ekacco puggalo ummujjati "sàdhu saddhà kusalesu dhammesu, sàdha hiri kusalesu dhammesu, sàdhu ottappaü kusalesu dhammesu, sàdhu viriyaü kusalesu dhammesu, sàdhu pa中à kusalesu dhammesu"tã. Tassa taü viriyaü neva hàyati, no vaóóhati, ñhità hoti, tassa sà pa中à neva hàyati, no vaóóhati, ñhità hoti. Evaü kho bhikkhave puggalo ummujjitvà ñhito hoti.

(4) Katha勺a bhikkhave puggalo ummujjitvà vipassati, viloketi: idha pana bhikkhave ekacco puggalo ummujjati "sàdhu saddhà kusalesu dhammesu, sàdhu hiri kusalesu dhammesu, sàdhu ottappaü kusalesu dhammesu, sàdhu viriyaü kusalesu dhammesu, sàdhu pa中à kusalesu dhammeså"ti. So tiõõaü saüyojanànaü parikkhayà sotàpanno hoti avinipàtadhammo niyato sambodhiparàyano. Evaü kho bhikkhave puggalo ummujjitvà vipassati. Viloketi.

(5) Katha勺a bhikkhave puggalo ummujjitvà patarati: idha bhikkhave ekacco puggalo ummujjati "sàdhu saddhà kusalesu dhammesu, sàdhu hiri kusalesu dhammesu, sàdhu ottappaü kusalesu dhammesu, sàdhu viriyaü kusalesu dhammesu, sàdhu pa中à kusalesu dhammeså"ti. So tiõõaü saüyojanànaü parikkhayà ràgadosamohànaü tanuttà sakadàgàmi hoti. Sakideva imaü lokaü àgantvà dukkhassantaü karoti. Evaü kho bhikkhave puggalo ummujjitvà patarati.

(6) Katha勺a bhikkhave puggaloummujjitvà patigàdhappatto hoti: idha bhikkhave ekacco puggalo ummujjati "sàdhu saddhà kusalesu dhammesu, sàdhu hiri kusalesu dhammesu, sàdhu ottappaü kusalesu dhammesu, sàdhu viriyaü kusalesu dhammesu, sàdhu pa中à kusalesu dhammeså"ti. So pa勺annaü orambhàgiyànaü saüyojanànaü parikkhayà opapàtiko hoti tatthaparinibbàyã anàvattidhammo tasmà lokà. Evaü kho bhikkhave puggalo ummujjitvà patigàdhappatto hoti.

Evaü kho bhikkhave puggalo ummujjitvà tiõõo hoti pàragato thale tiññhati bràhmaõo.
